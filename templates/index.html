Marvel Comics data analysis


Overview

The project's goal is walkthrough some skills learned at the bootcamp by analysis the characters of Marvel Comics.

Hypothesis: Machine learning models can be implemented in a systematically way...

Test hypothesis: We will implement a machine learning model step-by-step using generical functions.

For this project, We have used following technologies learned in this course: Python, Flask, Api requests, Data Analysis, Data Visualization and Machine Learning, among others.
