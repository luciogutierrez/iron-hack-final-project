Project name

Walking through a ML model's production line

Objective

Ironhack has provided us with a very important toolkit that we can use when working with ML models, for some of us, and this happened to me personally, I went from the problem of not having tools to having too many and not knowing which ones to use.

The objective of the project is to walk through the different processes involved in the generation of a machine learning model.

  1. Data Acquisition
    • Api Request
    • Download dataset
  2. Data Cleaning
  3. Feature Engineering
  4. Model Selection
  5. Hyperparameter Adjustments
  6. Deployment

Api Requests

In this section we will use python "requests" method to connect to the Marvel Api.
We will download information of our favorite super heroes which we will use to generate an online catalog to access the web page of each one of them.

Data Visualization

In this section we will create and interpret different types of visualizations to better understand the Marvel dataset we are working with.

Supervised ML model

In this section we will work on a supervised learning model to predict whether the alignment of a superhero is good or evil.
