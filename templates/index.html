Marvel Comics end-to-end ML model


Objective

The objective of the project is to implement an end-to-end ML model, since data acquisition to model validation, based on Marvel Comics characters.

For this purpose, we will develop the topics:
  1. Data preparation.
  2. Data visualization.
  3. ML model development step-by-step.

In this project we used the tools; Python, Flask, API requests, Data analysis, Data visualization, Machine Learning, among others.








1. Data preparation


In this section, we collect and prepare the data that we'll use in the model.

  1. Data acquisition
    • Marvel API https://developer-marvel.com/
    • Kaggle Marvel Superheroes datasets Kaggle-marvel-superheroes
  2. Data analysis
  3. Data wrangling
  4. Data cleaning
  5. Data storage
And we define two datasets, one for training and one for predictions.
