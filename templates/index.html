Marvel Comics end-to-end ML model


Overview


The project's goal is to implement a end-to-end ML model, based on the Marvel Comics Characters.

Hypothesis: Machine learning models can be implemented in a systematically way.

For this project I used Python, Flask, Api requests, Data Analysis, Data Visualization and Machine Learning, among others.
