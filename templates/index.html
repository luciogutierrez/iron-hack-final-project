Marvel Comics end-to-end ML model


Overview


The project's goal is to implement an end-to-end ML model, based on the Marvel Comics Characters, through the processes:
  1. Data preparation.
  2. Data visualization.
  3. Machine Learning Model development step-by-step.

For this project, we used the tools; Python, Flask, API requests, Data analysis, Data visualization, Machine Learning, among others.
